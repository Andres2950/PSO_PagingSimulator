# SO_Proyecto2

## Compilar proyecto

La primera vez, ejecutar desde la raíz:
```
cmake -B build -S .
```
Con eso genera los makefiles

Para compilar se ejecuta:
```
cmake --build build
```

La app se ejecuta con:
```
./build/app/app
```
